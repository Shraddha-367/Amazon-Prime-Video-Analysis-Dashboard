# Amazon Prime Video Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive Power BI dashboard that analyzes content trends on Amazon Prime Video. The goal is to uncover meaningful insights about content distribution, growth trends, and audience preferences.

---
## 🎯 Objective

The main objectives of this project are:

* Analyze content growth over the years
* Understand genre distribution and popularity
* Perform data cleaning and transformation for accurate insights
---

## 📊 Dashboard Highlights

* 📈 **Trend Analysis:** Shows how content has grown over time
* 🍩 **Genre Distribution:** Visual breakdown of content by genre
* 🎬 **Content Type Split:** Movies vs TV Shows comparison
* 🔍 **Interactive Filters:** Explore data dynamically by genre

---

## 📈 Key Insights

* 🚀 Content production has significantly increased after 2000
* 🎬 Movies dominate the platform compared to TV Shows
* 🎭 Drama and Comedy are the most popular genres
* 📊 Recent years show rapid growth in content addition

---

## 🧹 Data Cleaning & Transformation

* Removed columns with high missing values (~90%+) to ensure accuracy
* Split multi-value columns like **Genre** into structured format
* Extracted and separated **Duration** into numeric value and type
* Standardized inconsistent values (e.g., Season vs Seasons)

---

## 🛠 Tools & Technologies

* **Power BI** (Dashboard & Visualization)
* **Power Query** (Data Cleaning & Transformation)
* **DAX** (Calculated Measures)
